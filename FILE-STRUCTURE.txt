=====================================
QUICK START
=====================================

1. Activate theme in WordPress admin
2. Install & activate WooCommerce plugin
3. Run setup-dummy-products.php in browser
4. Set homepage in Settings > Reading
5. Create navigation menu
6. Visit homepage - Done! 🎉

Full instructions: See QUICKSTART.md
