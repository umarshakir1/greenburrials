# WordPress Block Editor - Quick Reference Guide

## Now Supported on Your Blog Posts

### Layout Blocks

#### 📐 Media & Text (Side-by-Side)
**Use this for:** Placing images next to text content
- Image can be on left or right
- Automatically stacks on mobile
- Perfect for creating the layout shown in your screenshot

**How to use:**
1. Click (+) to add block
2. Search for "Media & Text"
3. Upload/select image
4. Add text content
5. Use settings to switch image position

---

#### 📊 Columns
**Use this for:** Multi-column layouts
- Supports 2, 3, or 4 columns
- Each column can contain any blocks
- Responsive (stacks on mobile)

**How to use:**
1. Click (+) to add block
2. Search for "Columns"
3. Select number of columns
4. Add content to each column

---

### Alignment Options

#### ⬅️ Align Left
Float content to the left with text wrapping around it
- Great for images within paragraphs
- Text flows around the right side

#### ➡️ Align Right
Float content to the right with text wrapping around it
- Great for images within paragraphs
- Text flows around the left side

#### ⬛ Align Center
Center content on the page
- No text wrapping
- Content is centered horizontally

#### 📏 Align Wide
Wider than normal content (1200px max)
- Breaks out of standard content width
- Still has margins on sides

#### 📐 Align Full
Full viewport width
- Edge-to-edge content
- No side margins
- Great for full-width images or sections

## Tips for Best Results

### For Side-by-Side Layouts (Like Your Screenshot)
1. Use the **Media & Text** block
2. Upload your image (it will appear on the left by default)
3. Add your text content in the right panel
4. To switch sides: Click block → Settings panel → Toggle "Media on the right"

### For Multi-Column Content
1. Use the **Columns** block
2. Choose 2 columns for side-by-side content
3. Add any blocks inside each column (images, text, lists, etc.)

### For Floating Images
1. Add an **Image** block
2. Click the image
3. In the toolbar, click the alignment button
4. Choose "Align left" or "Align right"
5. Add paragraph blocks after - text will wrap around the image

### For Full-Width Images
1. Add an **Image** block
2. Upload a large image (at least 1920px wide)
3. In the toolbar, click "Align full width"
4. Image will span edge-to-edge

## Mobile Behavior

All blocks are automatically responsive:

- **Columns** → Stack vertically
- **Media & Text** → Image on top, text below
- **Aligned images** → Become full-width
- **Tables** → Become scrollable
- **Galleries** → Single column

## Common Issues & Solutions

### Issue: Alignment not showing
**Solution:** Make sure you've saved the post and cleared your browser cache (Ctrl+Shift+R)

### Issue: Full-width not working
**Solution:** The block must be at the root level, not nested inside a group or column

### Issue: Colors not applying
**Solution:** Use the color picker in the block settings panel, select from "Theme colors"

### Issue: Mobile layout broken
**Solution:** This is now fixed! All blocks automatically stack on mobile devices

## Testing Your Changes

1. **Edit an existing blog post** or create a new one

3. **Preview the post** to see the side-by-side layout
4. **Check on mobile** (use browser dev tools or actual device)
5. **Try different alignments** to see all options working
